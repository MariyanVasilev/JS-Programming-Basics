За Пушване в GitHub

Първо отваряме конзолата от папката в която са ни файловете, там е и папката ( .git ) 

След като сме правили промени

1. - Изпълняваме команда - git status - така проверяваме дали има направени промени които не са камитнати
	ако има такива файловете излизат в червено.
2, - Изпълняваме команда - git add -A - така адваме файловете в червено
3. - Изпълняваме команда - git status - за да проверим отново има ли фалове в червено , ако има значи 
	не сме изпълнили предната команда правилно, и следователно я изпълняваме отново ( git add -A )
4. - Ако всички файлове са в зелено изпълняваме команда - git commit -m "<some message here...>"
	Ако всичко е точно ще ни изпише че няма нищо за commit а има за push 

5. - Изпълняваме команда - git push origin main

И ако искаме отново правим проверка с git status
